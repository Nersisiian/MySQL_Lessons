# MySQL_Lessons

SQLLesson — это учебный проект для изучения и практики SQL-запросов. Проект включает примеры баз данных, запросов и упражнений для закрепления знаний.

---

## Возможности

- Практика создания и изменения таблиц (`CREATE`, `ALTER`, `DROP`)
- Работа с данными (`INSERT`, `UPDATE`, `DELETE`)
- Фильтрация и сортировка (`WHERE`, `ORDER BY`, `LIMIT`)
- Агрегатные функции (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)
- Соединения таблиц (`JOIN`)
- Подзапросы и вложенные запросы
